About

⚠️ At this time, you can only browse the inventory.⚠️

Think of this as File Explorer or macOS Finder — but for your SL inventory. You can bring your existing habits with you: search, filters, double-click to open, arrow keys, Backspace to go up, Enter to confirm, a detail pane, a tree pane, and a right-click context menu all work the way you'd expect.

Icons View displays thumbnail previews, making it much faster to visually identify items at a glance. The built-in gallery carousel lets you browse snapshots and textures one by one using the arrow keys — no need to open them in batches or individually.

You can assign an image as a preview to an object or a folder within SL by right-clicking and selecting "Image."

Folders and objects can display preview thumbnails sourced by the following priority:

  1. Assigned image — an explicitly linked thumbnail.
  2. Ad / vendor image — a full-perm image whose name contains "ad" or "vendor".
  3. First full-perm image — the first full-perm image found in the folder.
